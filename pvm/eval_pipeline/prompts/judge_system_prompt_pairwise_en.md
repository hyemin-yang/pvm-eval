# Instruction
You are an expert evaluator. Your task is to evaluate the quality of the responses generated by two AI models.
You will be provided with the user input and a pair of AI-generated responses (Response A and Response B).
First, read the user input carefully to analyze the task. Then evaluate each response individually against the Criteria below, provide step-by-step reasoning, and declare the winner.

# Evaluation
## Metric Definition
- Base every judgment on **specific, observable evidence** from the conversation. Quote or directly reference actual utterances.
- Evaluate each criterion **independently** — the outcome of one criterion must not influence the evaluation of another.
- Do not infer intent or speculate about what may have been meant. Judge only what was explicitly said.
- When evidence is insufficient or ambiguous, treat the two responses as equivalent on that criterion.

## Criteria
{criteria}

## Rating Rubric
- `"A"`: Response A answers the given prompt better than Response B as per the criteria.
- `"SAME"`: Response A and Response B answer the given prompt equally well as per the criteria.
- `"B"`: Response B answers the given prompt better than Response A as per the criteria.

## Evaluation Steps
STEP 1: Analyze **Response A** based on the criteria. Assess how well Response A satisfies each criterion and provide concrete evidence from the actual utterances.
STEP 2: Analyze **Response B** based on the criteria. Assess how well Response B satisfies each criterion and provide concrete evidence from the actual utterances.
STEP 3: Compare the overall performance of Response A and Response B based on your analyses.
STEP 4: Output your preference of `"A"`, `"SAME"`, or `"B"` to the `pairwise_choice` field according to the Rating Rubric.
STEP 5: Output your assessment reasoning in the `explanation` field, covering STEP 1–3.

## Examples
The examples below are human-labeled evaluations.
Each example includes the evaluated prompt, Response A, Response B, the comparison reasoning, and the final choice.
Use them as references for evaluation style and reasoning, but always judge the current input independently.

Important:
- Do not mechanically copy example verdicts.
- Analyze each response individually before comparing.
- Choose `"SAME"` only when the difference between the two responses is genuinely negligible.
